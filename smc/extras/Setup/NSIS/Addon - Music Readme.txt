Secret Maryo Chronicles Addon : Music

- Installation :

Zip Package :

Copy the data directory to the Secret Maryo Chronicles directory.
If you get asked about replacing files press OK or Yes.

Setup :

Start the Setup and select your Secret Maryo Chronicles directory
if the default installer directory is invalid.
